# CMPE258_DL_TrainingDesign

A structured deep learning tutorial repository covering the core ideas behind neural network training design, convolutional neural networks, optimization, tuning, and model evaluation.

This project is organized as a set of Jupyter notebooks, with each notebook focusing on one important area of deep learning. Together, they form a practical learning path from foundational concepts to modern architectures and performance evaluation.

Video Link: https://youtu.be/4apjV3xrIvY
---

## Overview

Deep learning performance depends on more than just building a model. Strong results come from understanding:

- the role of activation functions
- how convolutional neural networks work
- how optimizers affect convergence
- how hyperparameters influence training
- how to evaluate models using the right metrics
- how modern CNN architectures improve depth, efficiency, and learning stability

This repository is designed to walk through those topics step by step in a clear and practical way.

---

## Repository Structure

### 1. `final_activation_functions_tutorial.ipynb`
This notebook introduces activation functions and explains why nonlinearity is essential in neural networks. It helps build intuition for how different activation functions influence gradient flow, representation learning, and model behavior.

### 2. `final_cnn_fundamentals_tutorial.ipynb`
This notebook covers the fundamentals of convolutional neural networks from first principles. It explains convolutions, feature maps, pooling, CNN layers, and the motivation for using CNNs in image-related tasks.

### 3. `final_hyperparameter_tuning_tutorial.ipynb`
This notebook focuses on hyperparameter tuning and its role in improving training performance. It highlights how choices such as learning rate, batch size, and other tuning variables impact model quality and convergence.

### 4. `final_important_classification_metrics_tutorial.ipynb`
This notebook explains the most important classification metrics used in machine learning. It covers concepts such as confusion matrix, accuracy, precision, recall, F1-score, ROC curves, PR curves, multi-class evaluation, and imbalanced data settings.

### 5. `final_modern_cnn_architectures_tutorial.ipynb`
This notebook explores the evolution of CNN architectures from early models to more advanced designs. It discusses deeper and more effective architectures such as ResNet-style ideas and other modern CNN improvements that address training difficulty and efficiency.

### 6. `final_optimizers_deep_learning_tutorial.ipynb`
This notebook explains optimization methods used in deep learning training. It focuses on how optimizers affect convergence speed, stability, and final model performance.

## Learning Goals

By working through this repository, the learner should be able to:

- understand why activation functions matter in deep networks
- explain the key building blocks of CNNs
- compare optimization strategies for training neural networks
- understand the purpose of hyperparameter tuning
- evaluate classification models using appropriate metrics
- recognize the motivation behind modern CNN architecture design

## Why This Repository Matters

This repository treats deep learning as a full design pipeline rather than a collection of isolated experiments.

It starts with the foundations of neural networks, then moves into CNN mechanics, architecture design, training strategy, and finally model evaluation. That makes it useful both as a study resource and as a reference for understanding how different parts of the training process connect together.
